# 🤖 Doori (두리)

> **2025 지능형 로봇 분야 SDGs 아이디어 공모전 대상 (Grand Prize) 수상**
> 도시 안전 사각지대 해소를 위한 AI 스마트 순찰 로봇 프로토타입

---

## 🎬 프로젝트 개요

Doori(두리)는 용인시의 급증하는 포트홀 문제와 시각장애인을 위한 점자블록 파손 문제를 해결하기 위해 제안된 **AI 자율 순찰 로봇**입니다. 단순한 아이디어 제안을 넘어, 3D 모델링과 Unity 시뮬레이션을 통해 기술적 실현 가능성을 검증한 **프로토타입** 프로젝트입니다.

* **문제 정의**: 용인시 포트홀 발생 2년간 75.6% 급증, 경기도 점자블록 적합 설치율 단 4%
* **솔루션**: AI 기반 실시간 노면 탐지 및 지자체 관제 서버 연동 시스템
* **SDGs 연계**: SDG 9(산업혁신), SDG 10(불평등 감소), SDG 11(지속가능한 도시)

---

## ✨ 주요 기능

| 기능 | 상세 설명 |
| --- | --- |
| **실시간 위험 탐지** | 고해상도 카메라를 통해 포트홀, 파손 점자블록, 쓰러진 볼라드를 AI가 판별 |
| **자율 주행 (SLAM)** | 360° LiDAR 센서를 활용하여 장애물을 회피하고 최적의 순찰 경로 주행 |
| **데이터 자동 전송** | 위험 요소 발견 시 GPS 좌표와 현장 사진을 지자체 서버로 즉시 전송 |
| **정비 우선순위 분류** | 수집된 데이터를 바탕으로 긴급 정비가 필요한 구역을 Priority 1~3으로 분류 |

## 🛠️ 기술 구현

### 1. Unity 시뮬레이션

![Doori Simulation 데모](docs/dori.gif)

*로봇의 자율 주행 및 객체 탐지 알고리즘을 Unity 환경에서 시뮬레이션하여 논리를 검증하였습니다.*

### 2. 3D 로봇 설계

* **파일**: `model/Doori_Robot_Model.fbx`
* **특징**: 보도 단차 극복을 위한 무한궤도(Crawler) 방식 채택 및 센서부(LiDAR, Camera) 최적 배치 설계

### 3. 시스템 아키텍처 (Proposed)

* **AI Model**: YOLOv8 기반 객체 인식
* **Robotics**: ROS2 기반 제어 시스템 설계
* **Sim Engine**: Unity 2022.3.47f1
